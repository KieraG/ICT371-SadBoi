## Building
### Tools
* Unity 2019.3.7f1.
* Git LFS Required

To build, open the project in Unity Editor 2019.3.7f. 
* File -> Build And Run
* Build can be compiled for PC, Mac, & Linux
* Build can also be compiled for WebGL
